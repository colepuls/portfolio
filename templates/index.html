Projects

  • Pico

    Desktop Robot Assistant (Raspberry Pi 5, Arduino UNO R3)

    GitHub

    Pico is a fully offline personal robot built using Raspberry Pi and Arduino, focused on real time audio, vision, and control. I designed and deployed a custom always on wake word model in PyTorch, implemented face tracking with MediaPipe and OpenCV, and built a robust Pi to Arduino serial control system for motion. The system runs local command handling without cloud services and is packaged in a 3D-printed enclosure integrating camera, audio, servos, and a visual display.

    Python C++ OpenCV PyTorch Torchaudio

  • 2Will

    Two-Wheeled Self-Balancing Robot (ESP32)

    GitHub

    2Will is a two-wheeled self-balancing robot built around an ESP32, focused on real-time sensor fusion, control systems, and embedded motor control. I am implementing IMU-based state estimation using a 9-axis IMU and developing a PID control loop to maintain balance and stability. The system integrates motor encoders, a dual motor driver, and onboard power regulation, with modular firmware designed for iterative tuning and experimentation. The robot is being developed as a hands-on platform for learning balance control, embedded systems, and real-time feedback loops.

    C++ PlatformIO PID Control
